Event Registration Page Project
--------------------------------

Description:
This is a responsive HTML/CSS event registration form that allows users to input their details and view them on a confirmation page.

Files:
- register.html: Main registration form.
- confirmation.html: Displays user input after registration.
- css/style.css: Styling for the form, layout, and responsive behavior.
- README.txt: Project information.

Features:
- HTML5 form validation using 'required', 'type', and 'pattern' attributes.
- Responsive design for different screen sizes.
- Visual feedback upon submission.
- Pure HTML/CSS with a small JavaScript snippet to display submitted details.

Usage:
1. Open 'register.html' in your browser.
2. Fill out the form and submit.
3. You will be redirected to 'confirmation.html' showing your details.
